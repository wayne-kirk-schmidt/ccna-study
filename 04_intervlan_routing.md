# Inter-VLAN Routing

## 1. Overview
Inter-VLAN routing enables communication between VLANs using Layer 3 devices.

## 2. Goals for Mastery
- Explain why VLANs cannot communicate by default
- Understand router-on-a-stick and SVIs

## 3. Concept List
- Router-on-a-stick
- Switch Virtual Interfaces (SVI)
- Default gateway placement

## 4. Sample Problems / Questions
### Question 1
Why can hosts in different VLANs not communicate?

### Question 2
What device performs inter-VLAN routing?

### Question 3
Where is the default gateway configured?

## 5. Answers
### Answer 1
They are separate broadcast domains

### Answer 2
A router or Layer 3 switch

### Answer 3
On the Layer 3 interface or SVI
