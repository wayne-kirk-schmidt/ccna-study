# OSI & TCP/IP Models

## 1. Overview
The OSI and TCP/IP models provide a framework for understanding how data moves through a network and where failures occur.

## 2. Goals for Mastery
- Identify which protocols operate at which layer
- Diagnose problems by layer
- Understand encapsulation flow

## 3. Concept List
- OSI 7 layers
- TCP/IP 4-layer model
- Encapsulation
- Layer-based troubleshooting

## 4. Sample Problems / Questions
### Question 1
At which OSI layer does ARP operate?

### Question 2
Which layer is responsible for end-to-end reliability?

### Question 3
Why might DNS work but ping fail?

## 5. Answers
### Answer 1
Layer 2/3 boundary

### Answer 2
Transport layer

### Answer 3
ICMP may be blocked while DNS (UDP/53) is allowed
